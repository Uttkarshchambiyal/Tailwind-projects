# Profile Business Card

A clean and responsive profile business card UI built with HTML and Tailwind CSS.

## Preview

This project showcases a modern personal profile card for **Uttkarsh Chambiyal** with:
- Profile image
- Role and location
- Skill tags
- Short bio
- Stats section
- Social links
- Follow button
