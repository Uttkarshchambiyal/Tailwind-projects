## ✨ Tailwind Concepts Practiced

- `flex`, `justify-between`, `gap-*` for layout
- `bg-*`, `text-*` for color theming (blue, purple, amber palette)
- `text-[40px]` arbitrary values
- `hover:text-red-200` interactive states
- `leading-relaxed`, `max-w-2xl` for typography control
- `shadow-inner`, `font-medium` for visual depth
- `h-*`, `p-*`, `px-*` spacing utilities
