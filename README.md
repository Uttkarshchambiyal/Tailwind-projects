# 🎨 Tailwind Projects

A curated collection of responsive and modern UI components and web pages built using **Tailwind CSS**. This repo serves as a hands-on practice ground and portfolio showcase for Tailwind-based designs.

---

## 🚀 About

This repository contains multiple mini-projects and UI components created while learning and mastering **Tailwind CSS**. Each project focuses on clean, utility-first styling with a mobile-first responsive approach.

---

## 📁 Projects Included

| Project | Description |
|--------|-------------|
| Landing Page | Responsive hero section with navbar and CTA |
| Card Components | Profile cards, product cards with hover effects |
| Navbar | Sticky and responsive navigation bar |
| Forms | Styled login and signup forms |
| Dashboard UI | Clean admin-style dashboard layout |

> *(More projects added regularly)*

---

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS** (via CDN / CLI)
- **JavaScript** (where applicable)

---

## 📦 Getting Started

```bash
# Clone the repository
git clone https://github.com/Uttkarshchambiyal/Tailwind-projects.git

# Navigate to project folder
cd Tailwind-projects

# Open any project in your browser
open index.html
```
