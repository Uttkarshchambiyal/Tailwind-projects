## 🌊 About This Repo

> **A curated collection of responsive and modern UI components and pages built using Tailwind CSS.**
> This repo is a hands-on practice ground and portfolio showcase for Tailwind-based designs.
> Every component is **mobile-first**, **utility-first**, and production-ready.
